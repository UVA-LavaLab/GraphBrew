# Output Formats (strict)

Every deliverable must include:

---

## 1) Summary (5-10 lines)

- What you did
- What you found
- What changed (if any)

## 2) Evidence

- File paths + line numbers for key claims
- Logs / timings (if measured)

## 3) Recommendations

- Ordered by impact vs effort
- Each has: test plan + success criteria

## 4) Next Actions

- A short TODO list

---

## Naming

- Use `docs/agent_*.md` files for reports.
- Use `agent/notes/` for scratch notes if needed.
